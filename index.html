Abstract

Recent studies reveal that selecting informative and relevant video frames can significantly improve the performance of Video Large Language Models (Video-LLMs). Current methods are mostly training-free approaches, such as reducing inter-frame redundancy, employing separate models for image-text relevance assessment, or utilizing temporal video grounding for event localization, whereas they struggle to address the complex scenarios of real-world video understanding. To address these challenges, we propose Instructed Temporal Grounding for Videos, namely VideoITG, which features customized sampling aligned with user instructions. We first develop an automated data annotation pipeline that consists of three steps: detailed clip-level description generation, question-guided clip retrieval, and task-specific frame selection. With this pipeline, we create a VideoITG dataset containing 40K videos and 500K instruction-dependent key frame selection annotations. We then design a plug-and-play VideoITG model, which takes advantage of visual language alignment and reasoning capabilities of Video-LLMs, for effective video frame selection in a discriminative manner. Coupling with current Video-LLMs, VideoITG achieves consistent performance improvements across multiple multimodal video understanding benchmarks, showing its superiority and great potentials for video understanding. Data, model and code will be made publicly available.
